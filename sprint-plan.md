1. Product Backlog
Top 5 prioritized tasks as user stories:

Add New Task: As a user, I want to create tasks with title, description, and due date to organize my workload.
Assign Task to User: As a manager, I want to assign tasks to team members to distribute responsibilities.
Mark Task Complete: As a user, I want to update task status to completed for accurate tracking.
Display Task List: As a user, I want to view tasks filtered by status or assignee to monitor progress.
Notifications: As a user, I want alerts for assignments and deadlines to stay updated.

2. Sprint Goal
This sprint will implement core task creation and assignment functionality in a cloud prototype, enabling basic end-to-end workflows while demonstrating team Git collaboration.
